Project Overview
This repository includes three beginner-friendly JavaScript solutions that tackle common programming exercises:
Marks to Grade Converter
Requests a student’s score (0–100).
Determines and displays the grade based on the following rules:
A → Above 79

B → 60–79

C → 49–59

D → 40–49

E → Below 40

Vehicle Speed Checker
Reads the speed of a vehicle.
If the speed goes beyond the legal limit, it calculates and outputs demerit points for the driver.
Net Salary Calculator
Computes an individual’s net salary.
Inputs: basic salary and benefits.
Deductions applied: PAYE tax, NHIF, and NSSF.
